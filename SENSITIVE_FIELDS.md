# Sensitive Fields Registry

The following fields have been evaluated for sensitivity in accordance with data protection guidelines.

| Field Name | Sensitivity Level | Description | Rationale |
|------------|-------------------|-------------|-----------|
| `country` | Low | National identifier | Aggregated data |
| `year` | Low | Time reference | Public domain info |
| `prevalence`| Low | Estimated % | Synthetic value |
| `incidence` | Low | Estimated cases/year | Synthetic value |
| `n_cases` | Low | Total cases count | Aggregated/Synthetic |

**Conclusion**: This dataset contains no sensitive personal information or high-risk identifiers.
