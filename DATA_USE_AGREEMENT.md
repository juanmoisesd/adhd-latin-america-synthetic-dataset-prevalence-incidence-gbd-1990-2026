# Data Use Agreement (DUA)

## 1. Scope
This agreement applies to all users of the ADHD Latin America Synthetic Dataset (v1.2.0).

## 2. Permitted Use
The data is provided for research, educational, and commercial purposes. There are no restrictions on reuse, modification, or redistribution.

## 3. Responsible Use
Users are expected to:
- Cite the dataset appropriately.
- Avoid any attempts to re-identify individuals (though the data is synthetic).
- Report any discovered biases or errors to the authors.

## 4. No Support
The authors provide no formal support for the interpretation or use of the data, beyond the documentation provided in the repository.
