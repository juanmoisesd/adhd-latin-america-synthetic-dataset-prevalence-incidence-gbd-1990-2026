# Business Continuity Plan

## 1. Scope
Ensuring the availability of the ADHD Latin America dataset in case of service disruptions.

## 2. Infrastructure
- **Primary**: GitHub Repository.
- **Secondary**: Zenodo Archival.
- **Offline**: Local backups maintained by the authors.

## 3. Recovery Procedure
In the event of GitHub downtime, users can access the last stable version (v1.2.0) via the DOI 10.5281/zenodo.19145316.
