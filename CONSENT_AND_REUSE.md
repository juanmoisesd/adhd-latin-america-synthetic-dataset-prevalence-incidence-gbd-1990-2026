# Consent and Reuse Policy

## 1. Consent Framework
This project uses **synthetic data**. Therefore, individual human consent is not required for the creation, distribution, or reuse of this dataset.

## 2. Reuse Strategy
We encourage maximum reuse of the data for:
- Academic research and meta-analyses.
- Machine learning and epidemiological modeling.
- Educational and awareness purposes.

## 3. Mandatory Attribution
While not legally required by the CC0 license, we strongly request users to provide appropriate attribution using the citation metadata provided.
