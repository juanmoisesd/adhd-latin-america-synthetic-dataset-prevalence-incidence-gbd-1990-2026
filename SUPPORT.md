# Support

## 1. Getting Help
If you have questions about the dataset or need assistance with the framework, please:
- Open a GitHub Issue.
- Consult the `FAQ.md`.
- Reach out via the official website: https://juanmoisesdelaserna.es

## 2. Scope of Support
We provide best-effort support for data interpretation and repo technicalities. We do not provide clinical medical advice.
