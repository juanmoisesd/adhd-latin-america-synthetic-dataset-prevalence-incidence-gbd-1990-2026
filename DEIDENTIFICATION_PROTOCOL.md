# De-identification Protocol

This repository is built on a **fully synthetic** data generation model. No personal data was accessed or processed during the creation of this dataset.

## 1. Aggregation Strategy
Data is aggregated at the national level by country and year, preventing any granular analysis that could correspond to individual data points.

## 2. No Re-identification Risk
Because the data is generated via simulation, it is impossible to re-identify any individual from this dataset.

## 3. Review Process
The de-identification status of the repository is reviewed with every major release (vX.X.0).
