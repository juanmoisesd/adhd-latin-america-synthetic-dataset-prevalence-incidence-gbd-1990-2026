# Contributing Guidelines

We welcome contributions to improve the synthetic model or the repository framework.

## 1. Reporting Bugs
Use GitHub Issues to report discrepancies in data or code.

## 2. Feature Requests
Submit an issue describing the new country or variable you would like to see.

## 3. Pull Requests
Ensure that:
- Your changes pass the `smoke_test_v13.py`.
- You update the `CHANGELOG.md`.
- You maintain the repository standard (A-J).
