# Release Notes - v1.2.0 (2026-03-15)

This release represents a significant upgrade in repository standards, aligning with the "0.001% standard" for high-integrity research data management.

## Key Features
- **FAIR Compliance**: Maximum metadata interoperability using JSON-LD, DataCite, and RO-Crate.
- **Security**: Hacker-grade defensive architecture with manifest files, checksums, and signature placeholders.
- **Data Governance**: Explicit legal, ethics, and retention policies.
- **Provenance**: Forensically traceable data lineage from raw sources to analysis-ready datasets.
- **Validated Dataset**: Synthetic ADHD data for 20 countries (1990-2026) verified via smoke tests.
