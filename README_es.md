# TDAH en América Latina: Conjunto de Datos Sintéticos (1990-2026)

Este repositorio proporciona un conjunto de datos de investigación de alto estándar sobre la prevalencia, incidencia y número de casos de TDAH en América Latina. Sigue el "estándar del 0.001%" para la gestión de datos de investigación, asegurando la máxima FAIRness, procedencia y seguridad.

## Estructura del Repositorio
El repositorio está organizado en varios grupos funcionales:
- **A. Identidad y Citación**: READMEs, Resúmenes y metadatos de citación.
- **B. Legal y Ética**: Licencia, Términos de Uso y políticas de privacidad.
- **C. Metadatos FAIR**: Metadatos interoperables (DataCite, Schema.org, RO-Crate).
- **D. Datos y Semántica**: Datos brutos y limpios, libros de códigos y esquemas.
- **E. Procedencia**: Linaje e historial de transformaciones.
- **F. Calidad**: Informes de validación y pruebas de integridad.
- **G. Reproducibilidad**: Definiciones de entorno y ejemplos.
- **H. Seguridad**: Firmas criptográficas y atestaciones de la cadena de suministro.
- **I. Gobernanza**: Operaciones y registro de riesgos.
- **J. Automatización**: Flujos de trabajo de CI/CD.
