# Data Provenance Statement

## 1. Overview
This repository contains a synthetic dataset derived from the Global Burden of Disease (GBD) trends for ADHD in Latin America. The provenance of the data is tracked from the initial generation script to the final analysis-ready formats.

## 2. Chain of Custody
- **Creation**: Juan Moisés de la Serna Tuya (v1.0.0, 2024).
- **Transformation**: Refined structure and standard alignment (v1.2.0, 2025).
- **Validation**: Automated integrity checks performed by the v13 Framework.

## 3. Tooling
The data is generated and processed using Python 3.10+, with standard libraries for simulation (numpy, random) and data management (pandas).

## 4. Lineage Files
Detailed machine-readable lineage is available in the `lineage/` directory (YAML, JSON-LD, TTL formats).
